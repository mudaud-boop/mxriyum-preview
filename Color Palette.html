Color palette.
Built from the existing site palette. Nine hues, each with a 10-step ramp, plus semantic tokens. Click any swatch to copy its hex. Warm mauve remains the protagonist — tomato, butter, and berry give recipe content and seasonal accents somewhere to live.

The nine.

Core palette at a glance. These are the base hues — each is step 500 in its ramp.

Ramps.

Ten steps per hue (50–900), generated in OKLCH for perceptually even transitions. Use 50–200 for backgrounds, 500 for base, 700–900 for text & hover states.

Semantic tokens.

Named by role, not hue. These are what components should reference — if the base hue ever shifts, everything updates at once.

In context.

Real UI surfaces showing how the palette reads together. Hover the controls to swap palette combinations live.
